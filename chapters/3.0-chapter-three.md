## Part I: The Word

Text is not just "stored characters." In Python, strings are the primary interface between code and human consciousness. Every message, every document, every display — strings.

This half of the chapter explores how to **speak precisely** in code: manipulate text, clean it, format it, transform it.

---

## Part II: The Path

Until now, your code has been linear:

```
Line 1 → Line 2 → Line 3 → ...
```

But life is not linear. It presents **bifurcations** and **repetitions:**

- *If the pitch is a fastball, swing. Otherwise, wait.*
- *Practice until the motion is perfect.*

Control flow is how you teach code to **think** — not just execute, but decide, adapt, repeat.
