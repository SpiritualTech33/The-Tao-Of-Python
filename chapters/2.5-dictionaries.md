# 2.5 Dictionaries

## The Most Powerful Collection

A dictionary stores **key-value pairs.** It is the most powerful and most used collection in Python.

The metaphor: a dictionary is the universe's knowledge structure. Every concept (key) has a meaning (value), and you access the meaning by knowing the name. Like the cosmos itself — everything is connected by correspondence.

```python
player = {
    "name": "Cosmos",
    "position": "pitcher",
    "batting_avg": 0.342,
    "active": True,
    "stats": {"hits": 34, "at_bats": 100}  # nested dict
}
```

## Creating and Accessing

```python
# Literal syntax
profile = {"name": "Cosmos", "age": 25}

# dict() constructor
profile = dict(name="Cosmos", age=25)

# Access by key
print(profile["name"])         # "Cosmos"

# Safe access — returns default if key missing
print(profile.get("city", "unknown"))   # "unknown" — no KeyError

# Check if key exists
"name" in profile    # True
"city" in profile    # False
```

---

## Modifying Dictionaries

```python
player = {"name": "Cosmos", "avg": 0.342}

# Add or update
player["age"] = 25              # Add new key
player["avg"] = 0.350           # Update existing

# Remove
del player["age"]               # Remove key — KeyError if missing
removed = player.pop("avg")     # Remove and return value
player.pop("missing", None)     # Remove safely with default

# Merge another dictionary (Python 3.9+)
extra = {"team": "Eagles", "city": "Mexico City"}
player |= extra    # Update in place
merged = player | extra   # New merged dict
```

## Iterating

```python
stats = {"hits": 34, "at_bats": 100, "walks": 12}

# Keys
for key in stats:
    print(key)

# Values
for value in stats.values():
    print(value)

# Both — the most common pattern
for key, value in stats.items():
    print(f"{key}: {value}")

# All methods return view objects — they update dynamically
keys_view = stats.keys()
stats["rbi"] = 45
print(keys_view)   # dict_keys(['hits', 'at_bats', 'walks', 'rbi']) — includes new key
```

## Dictionary Comprehension

```python
words = ["tao", "zen", "void", "cosmos"]

# Map each word to its length
lengths = {word: len(word) for word in words}
# {"tao": 3, "zen": 3, "void": 4, "cosmos": 6}

# Filter and transform
players = [{"name": "Cosmos", "avg": 0.342}, {"name": "Miguel", "avg": 0.267}]
elite = {p["name"]: p["avg"] for p in players if p["avg"] >= 0.300}
# {"Cosmos": 0.342}
```

## setdefault and defaultdict

```python
# setdefault — set a key only if it doesn't exist
config = {}
config.setdefault("debug", False)    # Sets debug=False
config.setdefault("debug", True)     # Does nothing — debug already exists
print(config)   # {"debug": False}

# Count word frequencies — classic pattern
words = ["zen", "tao", "zen", "code", "tao", "zen"]
counts = {}
for word in words:
    counts[word] = counts.get(word, 0) + 1
# {"zen": 3, "tao": 2, "code": 1}
```
