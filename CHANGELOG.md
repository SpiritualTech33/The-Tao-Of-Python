## [1.0.0] - 2026-04-08

### Added

- Chapter 1 — The First Steps: Variables, types, strings, operators
- Chapter 2 — The Many and the One: Lists, tuples, sets, dictionaries
- Chapter 3 — The Word and the Path: String manipulation, control flow, loops
- Chapter 4 — When Action Learns to Repeat: Functions, args/kwargs, scope
- Chapter 5 — The Art of Falling: Error handling, exceptions, EAFP/LBYL
- Chapter 6 — Truth Put to the Test: pytest, assert, parametrize, fixtures
- 7 Katas per chapter (42 total executable tests)
- ZenCode philosophy integrated throughout all examples
- Project README with setup instructions and philosophy
- MIT and Apache 2.0 dual license
- Virtual environment configuration (.python-version, requirements.txt)
