# 3.1 Advanced String Manipulation

## Strings as Sequences

A string is an immutable sequence of Unicode characters. Because it is a sequence, it supports all the operations you already know from lists: indexing, slicing, iteration.

```python
word = "Python"

# Iteration
for char in word:
    print(char)    # P, y, t, h, o, n

# Enumerate
for i, char in enumerate(word):
    print(f"{i}: {char}")

# Check length
len(word)     # 6

# Membership
"y" in word   # True
"x" in word   # False
```

## Multiline Strings

For text that spans multiple lines, use triple quotes:

```python
haiku = """
Old pond —
a frog jumps in,
sound of water.
"""

sql = """
    SELECT name, avg
    FROM players
    WHERE active = TRUE
    ORDER BY avg DESC
"""

# Escape sequences in regular strings
tab = "column1\tcolumn2"    # \t → tab
newline = "line1\nline2"    # \n → newline
quote = "He said \"hello\""  # \" → literal quote
backslash = "path\\folder"   # \\ → literal backslash
```

## Raw Strings

When backslashes should be literal (file paths, regular expressions):

```python
# Regular string — backslash is escape character
path_regular = "C:\\Users\\cosmos\\Documents"

# Raw string — backslash is literal
path_raw = r"C:\Users\cosmos\Documents"

# Both produce the same string
print(path_regular == path_raw)   # True

# Essential for regex patterns
import re
pattern = r"\d{3}-\d{4}"    # Match phone number pattern
```

## String Searching

```python
text = "The Tao that can be named is not the eternal Tao"

# find() — returns index of first match, -1 if not found
text.find("Tao")           # 4
text.find("Tao", 5)        # 44  (start search from index 5)
text.find("xyz")           # -1

# index() — same but raises ValueError if not found
text.index("Tao")          # 4
# text.index("xyz")        # ValueError

# count() — how many times does it appear?
text.count("Tao")          # 2
text.count("the")          # 0  (case-sensitive)
text.lower().count("the")  # 2  (case-insensitive)

# Check starts/ends
text.startswith("The")     # True
text.endswith("Tao")       # True
```

## Text Cleaning Patterns

Real-world text is messy. These patterns appear constantly:

```python
raw = "  \n  cosmos de la cruz   \t  "

# Strip whitespace
clean = raw.strip()             # "cosmos de la cruz"

# Normalize internal whitespace
import re
normalized = " ".join(raw.split())   # "cosmos de la cruz"

# Remove specific characters
no_dots = "...hello...".strip(".")   # "hello"

# Replace multiple patterns
messy = "Hello,  World!  How  are  you?"
import re
clean = re.sub(r' {2,}', ' ', messy)   # Remove multiple spaces
```

## Checking String Content

```python
"cosmos123".isalnum()    # True  — only letters and digits
"cosmos".isalpha()       # True  — only letters
"12345".isdigit()        # True  — only digits
"COSMOS".isupper()       # True  — all uppercase
"cosmos".islower()       # True  — all lowercase
"Cosmos De La Cruz".istitle()  # True — title case
"   ".isspace()          # True  — only whitespace
```

These methods enable input validation without regular expressions — useful for quick checks.
